EasyAccess

Аннотация проекта

EasyAccess — это браузерное расширение для повышения доступности веб-сайтов для людей с ограниченными возможностями здоровья. Проект начался с идеи создания инструмента для 1С, но эволюционировал в универсальное решение для любых сайтов.

Расширение позволяет:

  • Настраивать визуальные параметры (размер шрифта, цветовая схема, контраст)
  • Использовать голосовое управление через Yandex SpeechKit
  • Автоматически заполнять формы с помощью Yandex OCR
  • Адаптировать сайты с запретами на копирование
  • Выделять и сохранять важные строки с возможностью стилизации
