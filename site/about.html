EasyAccess

Описание проекта

EasyAccess — это универсальное расширение, повышающее доступность веб-сайтов для людей с ОВЗ. Изначально мы ориентировались на интеграцию с 1С, но позже расширили идею до глобального решения.

Ключевые функции:

  • Изменение визуального стиля сайта (контрастность, шрифт, цветовые схемы).
  • Голосовой ввод с помощью Yandex SpeechKit.
  • Умная работа с документами (Yandex OCR).
  • Автоввод и выделение данных.
  • Функция фиксации строк на сайтах с сохранением в аккаунте.

Технологии

  • Frontend: HTML, CSS, JavaScript, Chrome Extension API
  • Backend: Python, Go
  • API: Yandex SpeechKit, Yandex OCR
